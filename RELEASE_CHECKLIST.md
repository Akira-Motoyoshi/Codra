# Codra 配布前チェックリスト

## 基本チェック

- [ ] `node --check app.js` が成功する
- [ ] `npm install` が成功する
- [ ] `npm test` が成功する
- [ ] `npm run screenshots` でREADME用画像を更新できる
- [ ] `npm run check` が成功する

## 画面チェック

- [ ] 初回ウェルカム導線が表示される
- [ ] デモモードを開始できる
- [ ] デモガイドが表示され、閉じられる
- [ ] 企業保存・企業比較が動作する
- [ ] ES作成、AI改善モック、面接メモが動作する
- [ ] データ管理、エクスポート、インポートが動作する
- [ ] Supabase移行プレビューが表示される
- [ ] デモデータ削除と確認モーダルが動作する
- [ ] READMEのスクリーンショットが最新画面と一致する
- [ ] スマートフォン幅で主要画面が崩れない

## データ・セキュリティチェック

- [ ] localStorageをリセットして初回状態を確認する
- [ ] デモデータ以外のユーザーデータを不用意に含めていない
- [ ] 個人情報、実ユーザーのES、面接回答、研究メモが含まれていない
- [ ] APIキー、アクセストークン、Supabase秘密鍵が含まれていない
- [ ] `.env`、`.env.local`、`node_modules`、テスト生成物がGit管理対象外になっている
- [ ] サンプル企業情報が公式情報ではないことをREADMEと画面で確認できる

## 共有URLチェック

- [ ] GitHub Pages、Netlify、またはVercelの共有URLで画面が開く
- [ ] CSS・JavaScript・スクリーンショットが相対パスで読み込まれる
- [ ] 共有URL上でデモを開始できる
- [ ] ページを再読み込みしてlocalStorageデータが保持される
- [ ] AI APIが未接続でもモックモードで動作する
- [ ] 共有URLを第三者端末またはシークレットウィンドウで確認する

## 現在の制約

- 保存先はブラウザlocalStorageのみ
- 認証、Supabase DB、公式情報同期は未接続
- AI APIはサーバー側プロキシを含め未接続
- 企業情報はデモ用サンプル
